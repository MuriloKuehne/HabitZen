# PWA Icons

This directory should contain the following icon files for PWA support:

## Required Icons

### Standard PWA Icons
- `icon-16x16.png` - 16x16px favicon
- `icon-32x32.png` - 32x32px favicon
- `icon-192x192.png` - 192x192px Android icon
- `icon-512x512.png` - 512x512px Android icon

### iOS Home Screen Icons
- `apple-touch-icon-120x120.png` - 120x120px (iPhone)
- `apple-touch-icon-152x152.png` - 152x152px (iPad)
- `apple-touch-icon-167x167.png` - 167x167px (iPad Pro)
- `apple-touch-icon-180x180.png` - 180x180px (iPhone Retina)

## Icon Design Guidelines

- Use a simple, recognizable design
- Ensure good contrast for visibility
- Use consistent branding colors
- Icons should work well at small sizes
- For iOS, icons are automatically rounded, so design should account for this
- Background should be solid color (not transparent) for best results

## Generation Tools

You can use tools like:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

Or create icons manually using design software like Figma, Adobe Illustrator, or Canva.

## Quick Setup

1. Create a base icon (1024x1024px recommended)
2. Generate all required sizes using one of the tools above
3. Place all generated icons in this directory
4. Ensure filenames match exactly as listed above
